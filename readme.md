- Análise de acompanhamento de vendas com SQL e Excel

- Objetivo
Criar um dashboard de vendas com os principais indicadores de desempenho e com os principais drivers dos resultados do mês

- Ferramentas utilizadas
- PostgreSQL
- pgAdmin
- Excel (dashboard interativo)

- Análises Envolvidas
- Receita:
- Calculo da receita total
- Avaliação do desempenho financeiro do período analisado

-Leads
- Análise de quantidade de leads gerados
- Relação entre geração de leads e vendas realizadas

- Top 5 marcas mais vendidas
- Identificação das marcas com maior volume de vendas
- Ranking das marcas mais performáticas

- Lojas que mais venderam
- Comparação de performance entre lojas
- Identificação das unidades com maior faturamento

- Visitas ao site por dia da semana
- Análise de comportamento dos usuários
- Identificação dos dias com maior tráfego

-Estados que mais venderam
- Análise geográfica das vendas
- Identificação dos estados com maior volume de receita

- Dashboard
O dashboard foi desenvolvido no Excel, contendo:
- Indicadores de desempenho (KPIs)
- Gráficos comparativos 
- Rankings
- Análises temporais e geográficas

- Sobre o projeto
Projeto desenvolvido com foco em prática de consultas SQL, análise exploratória de dados e construção de visualizações estratégicas para suporte a tomada de decisão.
